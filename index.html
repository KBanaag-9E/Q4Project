Projects/Accomplishments

Portfolio Overview
This portfolio consists of all the quarterly project assignments of each group member done throughout the entire Grade 9 school year.
Quarter 1 features layouts for individual online shops to practice the use of HTML and CSS.
Quarter 2 features responsive recipe-sharing websites where users can explore, view, and submit recipes for Filipino cuisine with the use of Bootstrap.
Quarter 3 features train fare guides for those using Filipino train stations.
Quarter 4 features webpages that calculates for a user's the Body Mass Index (BMI).
